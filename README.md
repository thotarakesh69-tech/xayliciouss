# Xayliciouss Premium Bakery Website

Xayliciouss is a premium homemade bakery and dessert website designed to provide a smooth online experience for customers to explore products, place orders, contact support, and interact with the brand.

## Live Demo

**Deployed Website:**
https://xayliciouss-meil.onrender.com

## GitHub Repository

https://github.com/thotarakesh69-tech/xayliciouss.git

## Project Overview

This project is developed for **Xayliciouss**, a premium homemade bakery brand offering brownies, cakes, cupcakes, dessert boxes, cookies, and customized desserts.

The website focuses on a warm, elegant, and luxury bakery experience with product browsing, cart functionality, WhatsApp ordering, customer support, sign-in, dashboard, and chatbot features.

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* Node.js
* Express.js
* JSON file-based database
* Render deployment
* GitHub
* WhatsApp integration
* Gemini-ready AI chatbot structure

## Main Features

* Premium responsive bakery website design
* Home, Menu, About Us, Reviews, FAQ, Contact, Sign In, Dashboard, and Customer Service pages
* Product listing with images, prices, descriptions, and categories
* Product search and category filtering
* Add to cart, quantity update, remove item, and total price calculation
* Sign-in required before placing an order
* WhatsApp checkout with pre-filled order details
* Floating WhatsApp button
* Customer dashboard with order tracking, order history, and bonus points
* Customer service form for enquiries, feedback, complaints, and custom cake requests
* AI chatbot popup for menu help, order guidance, and customer support
* FAQ accordion
* Customer reviews section
* Instagram-style gallery section
* Google Maps preview
* SEO-friendly structure
* Mobile and desktop responsive layout

## Problem It Solves

Many small homemade bakery businesses depend only on WhatsApp or Instagram for orders, which can make product browsing and customer communication difficult.

This website gives Xayliciouss a professional online presence where customers can:

* View bakery products clearly
* Search and filter products
* Add items to cart
* Place orders through WhatsApp
* Contact customer support
* Track account-related order details
* Get help through chatbot support

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/thotarakesh69-tech/xayliciouss.git
```

2. Open the project folder:

```bash
cd xayliciouss
```

3. Install dependencies:

```bash
npm install
```

4. Start the server:

```bash
npm start
```

5. Open in browser:

```text
http://localhost:3000
```
